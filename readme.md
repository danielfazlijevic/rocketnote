# RocketSync
### Synchronized note taking web app

## Demo
[https://danielfazlijevic.github.io/rocketnote/](https://danielfazlijevic.github.io/rocketnote/)


### Installation
Clone the repo
```
git clone https://github.com/danielfazlijevic/rocketnote.git
```
Install dependecies
```
npm install
```
Run gulp
```
gulp
```
